# Cash Time Machine

### Environment Setup

#### Windows

1. Create a virtual environment:
   ```
   python -m venv .venv
   ```
2. Activate the virtual environment:
   ```
   .venv\Scripts\activate
   ```
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

#### macOS/Linux

1. Create a virtual environment:
   ```
   python3 -m venv .venv
   ```
2. Activate the virtual environment:
   ```
   source .venv/bin/activate
   ```
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
